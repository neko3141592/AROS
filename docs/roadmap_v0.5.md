# ARL 開発ロードマップ v0.5: Web UI (Streamlit) と本番運用に向けた統合

バックエンドのループが完成したため、ユーザーが直感的にシステムの動きを監視・制御できるダッシュボードを構築し、システム総仕上げを行います。

## 1. Streamlitダッシュボード構築
- [ ] プロジェクトルートに `frontend/dashboard.py` を作成
- [ ] データベース（PostgreSQL）から現在進行中のタスク状態、過去の履歴を取得するAPI/データ層実装
- [ ] **UIコンポーネント**:
  - [ ] 新規タスク入力フォーム（プロンプト・要件入力）
  - [ ] LangGraphの現在地（実行中のノード）の可視化
  - [ ] 生成されたコードのハイライト表示画面
  - [ ] RunPodからのリアルタイムストリーミングログ表示（標準出力）
  - [ ] メトリクス推移のグラフ表示（Loss/Accuracy等）

- [ ] アーキテクチャ図とAPI仕様書の更新
